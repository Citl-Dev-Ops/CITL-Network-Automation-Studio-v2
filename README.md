# CITL Network Automation Studio v2.0

> **Shared Authorship** — CITL Development Team, Renton Technical College
> **Project Lead:** Doc McDowell

AV/IT operations, workstation diagnostics, display management, audio device configuration, and campus network tooling — portable suite for IT staff and students learning real-world sysadmin workflows.

## What It Does

- **AV/IT Operations** — HDMI switcher diagnostics, display port testing, projector control
- **Workstation Diagnostics** — save/restore display profiles, connection health checks, IT ticket generation
- **Audio Device Manager** — enumerate and configure audio endpoints, test mic/speaker chains
- **Network Diagnostics** — connection speed checks, DHCP validation, remote desktop testing
- **Quick-Fix Automation** — PowerShell-backed remediation scripts for common campus IT issues
